## 如何给node降级至14

要使用electron-app，需要把node降级至14.x。
【cursor推荐】使用nvm进行降级。降级成功的提示形如：`Now using node v14.17.0 (64-bit)`。
（nvm理应有自动将自己加入帐户环境变量的功能，原理类似gcc mingw64等东西。）
